# VR Molecular Chemistry Lab - Tec Task

---

## Unity Version
Unity 6 (6000.3.9f1)  
URP (Universal Render Pipeline)

---

## Requirements
- OpenXR enabled
- XR Interaction Toolkit
- Tested by XR Device Simulator

---

## How to Run
1. Download or clone this repository
2. Open Unity Hub
3. Click "Add" and select this project folder
4. Open using the Unity version mentioned above
5. Wait for packages to install
6. Open the scene: `GAME SCENE 1`
7. Click Play

---

## Features
- Atom grabbing system
- Molecule mixing logic
- Button press
- World Space Canvas always face camera ( UI) 
- Basic interaction using XR Toolkit
- Simple visual feedback for reactions

---

## Notes
This is a prototype project created as part of a task. Some parts are simplified and focused on interaction rather than full chemistry accuracy.
